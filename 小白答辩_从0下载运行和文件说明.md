# 小白答辩：从 0 下载、运行和认识项目文件

这份文档是给完全不熟悉代码的人看的。下午答辩时，不需要把每一行代码都讲懂，只要知道“这个项目有哪些实验、每个实验做什么、重要文件在哪里、怎么运行、结果在哪里”就可以。

## 一、这个仓库里主要有哪三个实验

本仓库可以按三个实验来讲：

```text
实验一：图书推荐系统实验
实验二：基于脑信号推断用户偏好的协同过滤推荐实验
实验三：基于脑信号推断用户偏好的深度学习推荐实验
```

简单理解：

1. 图书推荐系统实验：用图书评分数据做推荐。
2. EEG 协同过滤实验：把脑信号推断出的“喜欢/不喜欢”当成用户反馈，再用 UserCF 和 ItemCF 推荐。
3. EEG 深度学习实验：用 PyTorch 深度学习模型直接从 EEG 脑电信号中预测用户偏好，再生成推荐结果。

答辩时重点可以讲实验二和实验三，因为它们都围绕“脑信号推断用户偏好”。

## 三、如何安装运行环境

如果只是提交文件，不一定要运行。

如果老师要求现场运行，就需要安装 Python 环境。

### Mac / Linux

在终端进入项目根目录：

```bash
cd recommend-system
```

安装根目录图书推荐实验依赖：

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

进入 EEG 实验目录并安装 EEG 实验依赖：

```bash
cd eeg_preference_recommender
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

### Windows

在命令行进入项目根目录：

```bash
cd recommend-system
```

安装根目录图书推荐实验依赖：

```bash
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

进入 EEG 实验目录并安装 EEG 实验依赖：

```bash
cd eeg_preference_recommender
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

```text
recommend-system/eeg_preference_recommender/
```

这个文件夹是下午答辩最重要的部分，里面包含 EEG 协同过滤实验和 EEG 深度学习实验。

### `README.md`

EEG 推荐实验的总说明。

里面写了两个实验：

```text
实验一：基于脑信号推断用户偏好的协同过滤推荐
实验二：基于脑信号推断用户偏好的深度学习推荐
```

### `requirements.txt`

EEG 实验需要安装的依赖。

主要包括：

```text
numpy
pandas
scikit-learn
matplotlib
torch
```

简单说：

- numpy：做数字计算；
- pandas：处理表格数据；
- scikit-learn：计算相似度和评价指标；
- matplotlib：画图；
- torch：训练深度学习模型。

### `main.py`

EEG 深度学习推荐实验的一键运行入口。

它会自动完成：

1. 生成模拟 EEG 数据；
2. 提取 EEG 频段特征；
3. 训练深度学习模型；
4. 评估模型；
5. 生成推荐结果；
6. 生成图表和实验报告。

### `run_cf_experiment.py`

EEG 协同过滤推荐实验的一键运行入口。

它会自动完成：

1. 构造用户-物品偏好矩阵；
2. 计算用户相似度；
3. 计算物品相似度；
4. 生成 UserCF 推荐结果；
5. 生成 ItemCF 推荐结果；
6. 计算 Precision@10、Recall@10、HitRate@10；
7. 生成协同过滤实验报告。

### `build_preference_matrix.py`

构造 EEG 用户-物品偏好矩阵的脚本。

简单说：它把 EEG 偏好数据整理成一张大表：

```text
行：用户
列：物品
值：喜欢程度
```

喜欢可以记为 1，不喜欢可以记为 0，隐藏测试项保留为空。

### `collaborative_filtering.py`

协同过滤算法文件。

里面实现了：

```text
UserCF：基于相似用户推荐
ItemCF：基于相似物品推荐
```

它还会计算推荐评价指标。

### `src/`

深度学习 EEG 推荐实验的源码文件夹。

里面每个文件的作用如下：

```text

preprocess.py          对 EEG 信号做标准化和简单滤波
feature_extractor.py   提取 delta、theta、alpha、beta、gamma 频段特征
model.py               定义 EEGPreferenceNet 深度学习模型
train.py               训练深度学习模型
evaluate.py            评估模型，输出 Accuracy、Precision、Recall、F1、AUC 等指标
recommend.py           根据 EEG 偏好分数生成 Top-N 推荐结果
visualize.py           生成训练曲线、混淆矩阵、ROC 曲线、推荐柱状图等图片
report_generator.py    自动生成深度学习实验报告
```

## 八、答辩时可以怎么讲

可以按这个顺序讲：

1. 这个仓库里有三个推荐实验。
2. 第一个是传统图书推荐，用图书评分数据做推荐。
3. 第二个是 EEG 协同过滤推荐，把脑信号推断出的喜欢/不喜欢变成用户-物品矩阵，再做 UserCF 和 ItemCF。
4. 第三个是 EEG 深度学习推荐，用 CNN 模型从 EEG 信号中预测喜欢概率，再生成推荐列表。
5. 项目没有真实 EEG 采集硬件，所以使用模拟 EEG 数据验证完整算法流程。
6. 实验输出包括推荐结果、评价指标、可视化图片、报告和答辩 PPT。

## 九、老师常问问题和简单回答

### 问：为什么可以用模拟 EEG 数据？

答：

> 因为本实验主要验证算法流程，不涉及真实硬件采集。模拟数据参考 DEAP 情绪识别思路，用 valence 和 EEG 频段特征表示用户偏好，后续可以替换为真实 EEG 数据。

### 问：协同过滤实验和深度学习实验有什么区别？

答：

> 协同过滤实验是把 EEG 推断出的喜欢/不喜欢变成用户-物品矩阵，再根据相似用户或相似物品推荐；深度学习实验是直接把 EEG 信号输入 CNN 模型，让模型学习脑信号和用户偏好之间的关系。

### 问：UserCF 和 ItemCF 是什么？

答：

> UserCF 是找和目标用户兴趣相似的用户，再推荐这些相似用户喜欢的物品。ItemCF 是找和用户喜欢过的物品相似的其他物品，再推荐给用户。

### 问：深度学习模型做了什么？

答：

> 模型叫 EEGPreferenceNet，输入 EEG 脑电信号，经过 1D CNN 提取特征，输出用户喜欢该内容的概率和预测评分。

### 问：最终推荐分数怎么来的？

答：

> 深度学习实验里，推荐分数由 EEG 偏好分数和物品流行度融合得到：score = 0.7 × EEG 偏好分数 + 0.3 × 物品流行度。

### 问：项目有什么创新点？

答：

> 创新点是把 EEG 脑电信号作为用户隐式反馈。相比点击、评分、浏览记录，脑信号更接近用户即时的认知和情绪反应。
