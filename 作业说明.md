# 提交说明

本次提交的核心文件如下：

- `main.py`：图书推荐系统主程序
- `实验代码.py`：兼容运行入口
- `data/BX-Users.csv`
- `data/BX-Books.csv`
- `data/BX-Book-Ratings.csv`
- `README.md`
- `实验报告.md`
- `requirements.txt`

## 建议运行方式

```bash
.venv/bin/python main.py
```

## 程序功能

程序支持：

1. 读取 `BX-Users.csv`、`BX-Books.csv`、`BX-Book-Ratings.csv`
2. 进行基本数据清洗
3. 输出指定用户的图书推荐结果
4. 输出热门图书推荐结果
5. 把结果保存到 `output/demo_result.txt`

## 说明

为了方便答辩，默认直接运行时会自动选择一个演示用户，不需要临时手动找用户编号。
