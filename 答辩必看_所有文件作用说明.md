# 答辩必看：项目所有重要文件作用说明

这份文档只服务一个目的：下午答辩时不慌。

你不需要记住所有代码细节，只需要知道：

```text
1. 我这个项目有哪几个实验
2. 每个实验运行哪个文件
3. 每个重要文件是干什么的
4. 结果和报告在哪里
5. 老师问到时怎么解释
```

## 一、先记住一句总介绍

答辩开头可以这样说：

> 这个仓库主要包含三个推荐系统实验。第一个是基于图书评分数据的邻域协同过滤推荐；第二个是基于 EEG 脑信号偏好矩阵的协同过滤推荐；第三个是基于 EEG 脑信号特征的深度学习推荐。三个实验都是围绕“如何根据用户偏好生成推荐结果”展开的。

## 二、三个实验分别是什么

## 实验一：图书推荐系统实验

位置：

```text
recommend-system/
```

一句话解释：

> 这是一个传统图书推荐实验，使用 Book-Crossing 图书评分数据，根据用户之间的评分相似度给用户推荐图书。

它本质上属于：

```text
基于邻域的协同过滤推荐
```

更具体一点：

```text
User-Based Collaborative Filtering
基于用户的协同过滤
```

也就是说：

```text
先找和目标用户兴趣相似的用户
再把这些相似用户喜欢的书推荐给目标用户
```

## 实验二：EEG 协同过滤推荐实验

位置：

```text
recommend-system/eeg_preference_recommender/
```

一句话解释：

> 这个实验把 EEG 脑信号推断出的“喜欢/不喜欢”转成用户-物品偏好矩阵，然后用 UserCF 和 ItemCF 两种协同过滤算法生成推荐结果。

它和实验一的区别：

```text
实验一的数据来源：图书评分
实验二的数据来源：EEG 脑信号推断出的偏好
```

实验二的核心：

```text
脑信号偏好数据
↓
用户-物品偏好矩阵
↓
UserCF / ItemCF
↓
Top-N 推荐结果
```

## 实验三：EEG 深度学习推荐实验

位置：

```text
recommend-system/eeg_preference_recommender/
```

一句话解释：

> 这个实验使用 PyTorch 搭建 CNN 深度学习模型，直接从 EEG 脑电信号中预测用户是否喜欢某个内容，再根据预测偏好分数生成推荐结果。

实验三的核心：

```text
EEG 脑电信号
↓
深度学习模型 EEGPreferenceNet
↓
预测喜欢概率 preference_prob
↓
融合物品流行度
↓
Top-N 推荐结果
```

## 十、哪些文件不用讲

下面这些不用讲，也不用提交：

```text
.venv/
__pycache__/
.DS_Store
*.pyc
```

原因：

```text
.venv 是本地 Python 环境
__pycache__ 和 *.pyc 是 Python 自动生成的缓存
.DS_Store 是 macOS 自动生成的系统文件
```

它们不是实验内容。

## 十一、老师问到时怎么回答

## 1. 老师问：你这个项目到底做了几个实验？

回答：

> 主要做了三个实验。第一个是基于图书评分数据的邻域协同过滤推荐；第二个是基于 EEG 偏好矩阵的协同过滤推荐；第三个是基于 EEG 特征的深度学习推荐。

## 2. 老师问：第一个图书推荐实验是不是协同过滤？

回答：

> 是的。第一个图书推荐实验本质上是基于用户邻域的协同过滤推荐，也就是通过相似用户的评分记录给目标用户推荐图书。

## 3. 老师问：EEG 协同过滤和普通图书协同过滤有什么区别？

回答：

> 普通图书协同过滤使用用户评分数据，EEG 协同过滤使用脑信号推断出的喜欢或不喜欢作为隐式反馈，再构造用户-物品偏好矩阵进行推荐。

## 4. 老师问：EEG 深度学习实验和 EEG 协同过滤实验有什么区别？

回答：

> EEG 协同过滤实验先把偏好整理成用户-物品矩阵，再用相似用户或相似物品推荐；EEG 深度学习实验是直接把 EEG 信号输入 CNN 模型，让模型学习脑信号和用户偏好之间的关系。

## 5. 老师问：为什么没有真实 EEG 数据？

回答：

> 本实验重点是验证算法流程，不要求真实硬件采集，所以使用模拟 EEG 数据。模拟数据参考 EEG 情绪识别思路，包含 valence、arousal、preference_label 和 EEG 频段特征。后续可以替换成 DEAP 或 SEED 等真实 EEG 数据集。

## 6. 老师问：深度学习模型叫什么？

回答：

> 模型叫 EEGPreferenceNet，在 `src/model.py` 中定义，主要使用 1D CNN 提取 EEG 时间序列特征，输出用户喜欢该内容的概率。
